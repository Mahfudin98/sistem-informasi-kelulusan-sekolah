Panduan Instalasi & Penggunaan

Terima kasih telah menggunakan App Kelulusan. Ikuti panduan di bawah ini untuk memulai instalasi di server lokal maupun online.

1 Persyaratan Sistem

  • ✓
    PHP 8.2 atau lebih baru: Pastikan ekstensi pdo_mysql, gd, dan mbstring aktif.
  • ✓
    MySQL / MariaDB: Versi 5.7 atau lebih baru.
  • ✓
    Composer: Untuk manajemen dependensi.
  • ✓
    Apache / Nginx: Pastikan mod_rewrite aktif untuk mendukung routing.

2 Langkah Instalasi

Step 1: Install Dependensi

Buka terminal di folder project dan jalankan:

composer install

Step 2: Konfigurasi Environment

Copy file .env.example menjadi .env dan sesuaikan data berikut:

  • DB_NAME, DB_USER, DB_PASS (Kredensial database Anda).
  • MAIL_USERNAME, MAIL_PASSWORD (Untuk fitur lupa password).

Step 3: Setup Database

Jalankan script berikut untuk membuat tabel dan akun admin awal:

php database/setup_db.php

Step 4: Konfigurasi Virtual Host (Produksi)

Pastikan Document Root server Anda mengarah ke folder public/, bukan ke folder utama project.

3 Penggunaan Admin

Akses dashboard melalui URL /login dengan kredensial default:
Username: admin | Password: admin123

Import Data Siswa

Gunakan fitur Import Excel. Download template yang disediakan, isi data siswa, lalu upload kembali.

Atur Tanggal Pengumuman

Pengumuman akan otomatis terkunci (hitung mundur) sebelum tanggal yang ditentukan di Profil Sekolah.

🎓

⚠️ Tips Penting

  • Keamanan: Segera ubah password admin default dan tambahkan admin baru jika diperlukan.
  • Backup: Lakukan backup database secara rutin melalui menu Backup Database di dashboard superadmin.
  • Logo: Gunakan gambar logo sekolah dengan latar belakang transparan (PNG/SVG) untuk hasil terbaik.
